Способы вставить svg в html

  1. Через тег img
  2. Напрямую запихнуть svg в html
  3. Через спрайт

Способ 1

Минусы:

  1. Нельзя задавать свойства для svg, кроме width и height

Плюсы:

  1. Удобно и компактно в html-коде
  2. Не нужно изменять svg для того, чтобы заработало

Способ 2

Минусы:

  1. Допустим, эта иконка используется в 20 местах. А теперь подумай, что нужно сделать, чтобы поменять эту иконку во всех местах.
  2. Если нужна эта же иконка, но в другом, например, цвете, придется создавать отдельный файл под это. Например, box.svg и box-red.svg
  3. Сильно засоряется html-код из-за атрибута d у тега path

Плюсы:

  1. Не нужно изменять svg для того, чтобы заработало
  2. Можно задавать все доступные свойства для svg

Наведи на пример:

Значение currentColor у атрибута fill позволяет брать цвет из свойства color в css

Способ 3

Минусы:

  1. Нужно изменять svg для того, чтобы заработало
  2. Нужно использовать какой-то use
  3. Не работает без live Server ?
  4. Не поддерживает stroke-dasharray и stroke-offset

Плюсы:

  1. Поменял в одном месте - поменялось везде
  2. Если нужна эта же иконка, но в другом цвете, можно добавить дополнительный класс и по нему изменять цвет
  3. Можно задавать все доступные свойства для svg
  4. Не засоряется html-код
